# Analysis Scope

This repo is intended to hold the earlier hypertension analysis block comparing:

- **no follow-up**
- **has follow-up**

This is the first-phase work that produced:

- baseline comparisons
- retention summaries
- blood pressure summary figures
- patient journey figure
- Kaplan-Meier style follow-up analyses
- forest plot output

It does **not** currently aim to package the later visit-only GBTM trajectory work.
